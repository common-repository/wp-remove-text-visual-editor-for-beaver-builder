== Description ==

A Simple plugin to protect the work you have done on Beaver Builder by removing the visual and text interface on the post & page edit pages. 
It require you to obviouslly have Beaver Builder installed, and Disable Gutenberg plugin. 

Tested & stable with WordPress 5.2.3


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. It's already working, Enjoy.

== Frequently Asked Questions ==

= Will it work without Beaver Builder =

No.

= Will it work without Disable Gutenberg =

Yes, as long as your Beaver Builder is working (for example on older version without Gutenberg)
